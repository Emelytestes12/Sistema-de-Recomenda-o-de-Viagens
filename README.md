# Sistema-de-Recomenda-o-de-Viagens

Projeto em **Python** que recomenda um destino de viagem com base nas preferências do usuário, como clima, tipo de local e orçamento disponível.

---

## O que o código faz

* Define uma classe para representar destinos de viagem
* Armazena uma lista de destinos disponíveis
* Coleta preferências do usuário via terminal
* Compara preferências com os destinos cadastrados
* Retorna uma sugestão compatível ou informa se não houver resultados
